แผนผังระบบเครือข่าย (Network Topology)

สรุปภาพรวมระบบเครือข่ายอาคารวิศวกรรมศาสตร์

ระบบเครือข่ายออกแบบเป็นรูปแบบ Star Topology โดยมี Core Switch เป็นศูนย์กลาง และใช้ Fiber Optic SM 2Core เป็น Backbone หลักในการส่งสัญญาณไปยังแต่ละชั้น

แต่ละชั้นติดตั้ง Access Switch ภายใน ตู้ Rack 9U เพื่อกระจายสัญญาณให้กับอุปกรณ์ภายในชั้น เช่น คอมพิวเตอร์, Access Point และอุปกรณ์ไร้สาย โดยภายในตู้มีอุปกรณ์สำคัญ ได้แก่ Switch Cisco 24 Port, UPS APC On-Line 1000VA (Rack) และ PDU U9

ระบบยังมี Router สำหรับเชื่อมต่อกับอินเทอร์เน็ตภายนอก และ Server ที่ให้บริการระบบภายใน การออกแบบนี้ช่วยให้เครือข่ายมีความ เสถียรขยายได้ง่าย และสามารถแยกจุดผิดพลาดได้รวดเร็ว

สรุป: ระบบรองรับการใช้งานพร้อมกันของผู้ใช้จำนวนมาก ครอบคลุมทั้งอาคาร และพร้อมสำหรับการขยายในอนาคต

โครงการติดตั้งระบบเครือข่ายอินเทอร์เน็ตภายในอาคารคณะวิศวกรรมศาสตร์ ถูกออกแบบและดำเนินการภายใต้หลักการสำคัญ ได้แก่ ความเสถียร ความปลอดภัย และความสามารถในการขยายระบบในอนาคต โดยในทุกชั้นของอาคารได้ติดตั้งอุปกรณ์เครือข่ายตามมาตรฐานสากลเพื่อรับประกันการเชื่อมต่อที่มีประสิทธิภาพและต่อเนื่อง

ภาพรวมการติดตั้งประกอบด้วยตู้ Rack ขนาด 9U จำนวนทั้งสิ้น 17 ตู้ ซึ่งกระจายตัวเพื่อรองรับการเชื่อมต่อจาก Core Switch กลางไปยัง Access Switch แต่ละชั้น โครงข่ายหลักใช้สายไฟเบอร์ออปติกชนิด SM 2Core Flat Type เป็น Backbone ในการส่งสัญญาณระหว่างศูนย์กลางและจุดเชื่อมต่อภายในอาคาร เพื่อให้ระบบมีความเร็วและความเสถียรสูงสุดในการใช้งานจริง

งบประมาณรวมของการติดตั้งทั้งหมดอยู่ที่ 569,173 บาท (รวมค่าแรงและอุปกรณ์ทั้งหมด)ซึ่งถือเป็นการลงทุนที่คุ้มค่าเมื่อเทียบกับประสิทธิภาพของระบบเครือข่ายที่ได้ — ระบบนี้สามารถรองรับการใช้งานพร้อมกันของผู้ใช้หลายร้อยคนทั่วอาคาร โดยยังคงความเร็ว เสถียรภาพ และความปลอดภัยในระดับสูง

ระบบเครือข่ายนี้จึงนับเป็นโครงสร้างพื้นฐานสำคัญของอาคารวิศวกรรมศาสตร์ ที่ไม่เพียงตอบโจทย์การใช้งานปัจจุบันเท่านั้น แต่ยังรองรับการพัฒนาเทคโนโลยีใหม่ในอนาคตได้อย่างสมบูรณ์แบบ
